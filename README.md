## What I Learned | สิ่งที่ได้เรียนรู้

### Lab 1 - R Basics | พื้นฐาน R

- Create vectors, matrices, and arrays | สร้าง vectors, matrices, arrays
- Import CSV files | นำเข้าไฟล์ CSV
- Handle missing values (NA) | จัดการค่า missing (NA)

### Lab 2 - Visualization | การสร้างกราฟ

- Boxplot, Scatter plot | กราฟ Boxplot, Scatter plot
- Adding regression lines | เพิ่มเส้น regression
- Customizing themes | ปรับแต่ง theme

### Lab 3 - Statistics | สถิติ

- Chi-square test | การทดสอบ Chi-square
- Fisher's exact test | การทดสอบ Fisher
- Grouped bar charts | กราฟแท่งแบบ grouped

### Lab 4 - Data Wrangling | การจัดการข้อมูล

- filter, mutate, arrange, slice
- Pipe operator (%>%)
- Group by and summarise
